Questão relacionada a excluir o serviço


Por questões de boas práticas, o cadastro realizado não será excluido, só colocará no modo inativo. Sendo 1-ativo e 0-inativo, pois gera
segurança para o usuário do sistema caso tenha um deslize, podendo ser reencontrado novamente esse dado.
Porém, para fins educacionais, existe a função de realmente excluir o cadastro, para demonstrar que foi aprendido em sala e que funciona.
